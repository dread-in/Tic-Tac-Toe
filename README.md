# Hand-Drawn Tic-Tac-Toe ✏️⭕❌

A lightweight, fully functional Tic-Tac-Toe game with a nostalgic, sketchy aesthetic. I built this to challenge myself to create a complete game—including the user interface, styling, and game logic—entirely within a **single HTML file** using pure Vanilla web technologies.

## ✨ Features

*   **Custom Aesthetic:** The UI is designed entirely with custom CSS to look like a hand-drawn sketch on a piece of paper.
*   **Distinct Markers:** The 'X' and 'O' markers use distinct colors (red and blue) so the board is easy to read at a glance.
*   **Two Game Modes:** 
    *   **2-Player (Local):** Play against a friend on the same screen.
    *   **Bot Mode:** Challenge the built-in bot for a single-player experience.
*   **Zero Dependencies:** No external libraries or frameworks were used. The logic, styling, and markup are 100% self-contained.
